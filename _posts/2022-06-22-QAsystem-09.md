# 知识网络(Knowledge Graph)
为了能够更好的搜索文本，我们将会把pdf文档转换为text存入数据库中。这个数据库就是以图为拓扑结构的知识网络数据库。建立知识图谱的过程参考了Pelhans同学在知乎上[发表的文章](https://zhuanlan.zhihu.com/c_1018901137012928512)。

在Ubuntu下使用下列命令安装数据库：
```
sudo apt-get install mysql-server
sudo apt install mysql-client
sudo apt install libmysqlclient-dev 
```
