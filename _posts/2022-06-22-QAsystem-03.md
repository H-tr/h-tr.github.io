# 使用GPT3做简单应用并实现简易问答系统
为了简单方便，我们尝试GPT3的时候可以直接去找别人搭好的[playground](https://github.com/shreyashankar/gpt3-sandbox)。
将gpt3-sandbox下载下来之后我们选择其中的`gpt.py`来辅助我们的开发。
我们假设你现在在linux环境下开发（使用Windows的同学可以尝试WSL, Windows Subsystem Linux），
在terminal中输入：
```
mkdir GPTtry
```
建立一个文件夹作为试验田，体验GPT3的使用。在文件夹里，将gpt3-sandbox拷贝下来：
```
git clone https://github.com/shreyashankar/gpt3-sandbox.git
```
把其中的`gpt.py`留下，其余全都删除。

使用GPT的时候需要OpenAI给的secret key，这个上OpenAI的官网注册账号，然后就能拿到。
建立一个GPT_SECRET_KEY.json文件，拿到secret key以后将其储存在json文件中方便我们调用：
```json
{"API_KEY":"your-gpt-secret-key"}
```
用你自己获得的SECRET KEY代替your-gpt-secret-key。然后新建一个文件`GPT3.py`，用任何
你喜欢的编辑器打开，在里面加入代码。

首先我们获取需要的package和secret key：
```python
import json
import openai

with open('GPT_SECRET_KEY.json') as f:
    data = json.load(f)

openai.api_key = data['API_KEY']
from gpt import GPT
from gpt import Example
```
然后建立文本处理引擎：
```python
gpt = GPT(engine="davinci",
          temperature=0.5,
          max_tokens=100)
```
现在我们希望给这个语言模型一些范例，让它能够按照我所希望的样子工作。我们来给这个语言模型
一些自然语言，并给它有相同语义的sql语句。我们希望它能够从这些“自然语言-sql语言”对中找到
合适的转换逻辑，然后它能够帮我们转换更多更通用的自然语言指令。
```python
gpt.add_example(Example("Fetch unique value of DEPARTMENT from Worker table.",
                        "Select distinct DEPARTMENT from Worker;"))

gpt.add_example(Example("Print the first three characters of FIRST_NAME from Worker table.",
                        "Select substring(FIRST_NAME, 1, 3) from Worker;"))

gpt.add_example(Example("Find the position of the alphabet ('a') in the first name column 'Amitabh' from Worker table.",
                        "Select INSTR(FIRST_NAME, BINARY'a) from Worker where FIRST_NAME = 'Amitabh';"))

gpt.add_example(Example("Print the FIRST_NAME from Worker table after replacing 'a' with 'A'.",
                        "Select CONCAT(FIRST_NAME, ' ', LAST_NAME) AS 'COMPLETE_NAME' from Worker;"))

gpt.add_example(Example("Display the second highest salary from the Worker table.",
                        "Select max(Salary) from Worker where Salary not in (Select max(Salary) from Worker);"))

gpt.add_example(Example("Display the highest salary from the Worker table.",
                        "Select max(Salary) from Worker where Salary not in (Select max(Salary) from Worker);"))

gpt.add_example(Example("Fetch the count of employees working in the department Admin.",
                        "SELECT COUNT(*) FROM worker WHERE DEPARTMENT = 'Admin';"))

gpt.add_example(Example("Get all details of the Workers whose SALARY lies between 100000 and 500000.",
                        "Select * from Worker where SALARY between 100000 and 500000;"))

gpt.add_example(Example("Get Salary details of the Workers.",
                        "Select Salary from Worker;"))
```
现在让我们来看一下它的表现，给它一些测试数据：
```python

prompt = "Display the lowest salary from the Worker table."

```

现在我们熟悉了GPT3的调用，接下来试着完成一个简易的问答系统吧！

在OpenAI的网站上，有一个GPT3的playground。在这里我们可以调整参数，从而对与训练语言模型
进行微调。微调之后可以直接导出一段代码来设定python程序中语言模型的超参数。
我在其中已经尝试了一些调参，得到了一个我个人比较喜欢的结果，也希望大家能够自己玩一玩，
得到自己想要的模型。现在我们先建立一个文件
```
touch QAsystem.py
```
在这个文件中调取package和secret key并导入GPT3的超参数：
```python
import json
import openai

with open("GPT_SECRET_KEY.json") as f:
    data = json.load(f)

openai.api_key = data["API_KEY"]

def gpt3(prompt, engine='text-davinci-002', response_length=64,
         temperature=0.7, top_p=1, frequency_penalty=0, presence_penalty=0,
         start_text='', restart_text='', stop_seq=[]):
    response = openai.Completion.create(
        prompt=prompt + start_text,
        engine=engine,
        max_tokens=response_length,
        temperature=temperature,
        top_p=top_p,
        frequency_penalty=frequency_penalty,
        presence_penalty=presence_penalty,
        stop=stop_seq,
    )
    answer = response.choices[0]['text']
    new_prompt = prompt + start_text + answer + restart_text
    return answer, new_prompt
```
然后创建一个能够与我们对话的机器人：
```
touch chatbot.py
```
在这个python源代码中输入对话的引用函数：
```python
from QAsystem import gpt3

def chat():
    prompt = """Human: Hey, how are you doing?
AI: I'm good! What would you like to chat about?
Human:"""
    while True:
        prompt += input('You: ')
        answer, prompt = gpt3(prompt,
                              temperature=0.9,
                              frequency_penalty=1,
                              presence_penalty=1,
                              start_text='\nAI:',
                              restart_text='\nHuman: ',
                              stop_seq=['\nHuman:', '\n'])
        print('GPT-3:' + answer)


if __name__ == '__main__':
    chat()
```
其中prompt是传入语言模型的初始对话，它可以让语言模型知道接下来应该
按照什么样的模式进行对话和问答。然后我们每次输入的话都应该不断地添加在
prompt后面，这样语言模型才能够记住之前有过什么对话，从而不会出现明显的
割裂和违和感。

来运行一下问答系统吧：
```
python chatbox.py
```
这个系统会和使用者互动沟通，我们可以问他一些简单的问题，一段示例交流如下：
```
You: What is python?
GPT-3: Python is a programming language.
You: Do you know how to code?
GPT-3: Yes, I know how to code.
You: How to code?
GPT-3: In Python, the most basic way to code is to use the print statement.
You: Which language should I learn first?
GPT-3: I think it depends on what you want to do with coding. For example, if you want to make websites, you might want to learn HTML and CSS first.
```
可以明显地感觉到，这个问答系统已经初步知道如何应对追问，并且能更加深入的探讨某一种事物了。
