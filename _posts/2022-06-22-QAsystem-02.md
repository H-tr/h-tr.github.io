# 自然语言处理与问答系统
## 自然语言处理
在问答系统中，电脑需要能够“理解”文档的文本内容，所以它需要使用一些自然语言处理的手段。
NLP(natural language processing)中有两个state of art的模型可以帮助我们分析理解文本
语义，它们分别是OpenAI的GPT3和Google的BERT。

这两个模型是“预训练模型”，它们在大量的文本上进行了“补全故事”和“完形填空”的训练，能够根据
上下文的语境预测我们最想要什么样的词，然后把这个词汇填充在需要的地方，最终形成一段
完整的句子甚至文章。

训练这种语言模型需要大量的时间和庞大的算力，我们的个人电脑几乎无法做到这一点。幸运的是
OpenAI和Google已经帮我们做完了预训练，我们现在只需要下载使用预训练，甚至直接在Python中
调用相应的库就可以简单轻松愉快的使用这些语言模型了。
## 问答系统中的nlp
NLP的应用十分广泛，问答系统是其中十分重要的应用。因而很多公司和社区都专门为问答系统开发了
api（程序接口）。我们用的BERT是Huggingface公司开发的transformers package。在我们刚刚
配置好的环境中运行：
```
pip install transformers
```
即可下载transformers。若使用GPT3，则运行：
```
pip install openai
```
我们会分别使用两种语言模型实现一些示例，然后用它们分别搭建两个简易的通用问答系统（没有
专门的知识支撑）。然后我们会选用BERT作为进一步开发的模型。

感兴趣的同学可以自己试着实现一遍GPT3的开发，欢迎补充分享。
