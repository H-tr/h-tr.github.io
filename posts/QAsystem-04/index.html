使用BERT做简单应用并实现简易问答系统

上一章我们探讨了如何调用GPT3以及如何使用GPT3来搭建问答系统。现在让我们来简要介绍一下如何使用BERT搭建问答系统吧。

GPT3的训练使用的是文本预测的手段，即给定大量的文本，然后不断的让模型分析预测这个文本下一个单词、下一句话是什么，然后再根据原文本进行修正，这样不断的训练最终达到了非常强大的自然语言处理的能力。BERT使用的是另一种手法，即“完形填空”。它并不是单纯的预测后面的文本是什么样子的，而是把文本中其中一些词去掉，让模型预测去掉的词汇。这样的好处是语言模型可以有效合理的利用上下文的信息。

在尝试BERT的时候我们使用huggingface的transformers库。这里我们需要给定段文字，然后让BERT从这段文字中学习理解知识，然后回答我们的问题。

from transformers import pipeline

question_answering = pipeline("question-answering")
context = """
Machine learning (ML) is the study of computer algorithms that improve automatically through experience. It is seen as a part of artificial intelligence. Machine learning algorithms build a model based on sample data, known as "training data", in order to make predictions or decisions without being explicitly programmed to do so. Machine learning algorithms are used in a wide variety of applications, such as email filtering and computer vision, where it is difficult or unfeasible to develop conventional algorithms to perform the needed tasks.
"""

question = "What are machine learning models based on?"

result = question_answering(question=question, context=context)

print("Answer: ", result['answer'])
print("Score: ", result['score'])

其中打印出来的Answer就是模型给出的答案，Score就是这个答案的分数。context是BERT学习使用的文本，这个例子中是一段介绍machine learning的。

这是个异常简洁优雅的实现，它依赖于huggingface给我们提供的福利：pipeline package。根据hugging face官网上的介绍：

The pipelines are a great and easy way to use models for inference. These pipelines are objects that abstract most of the complex code from the library, offering a simple API dedicated to several tasks, including Named Entity Recognition, Masked Language Modeling, Sentiment Analysis, Feature Extraction and Question Answering. See the task summary for examples of use.

在pipelines里封装了很多复杂的代码，使得我们能够直接把BERT模型像api一样调用。如果只是想要单纯的实现问答系统的功能的小伙伴在这里就可以停下了。后面我们会稍微详细的介绍一下如果不用这个function，应该怎么实现BERT问答系统。

这个数据集中储存了大量的问题和回答，能够让语言模型知道自己应该是用什么样的方式回答问题。就像我们高考训练的时候做语文的阅读理解，仅仅能够理解文章是不行的，如果卷子上问“你是否赞同作者的观点”，回答一个“是”或“否”可能只能得一分，甚至一分都不得。这个时候我们是怎么知道应该如何答题呢？那就是老师会给我们举例子做讲解。有了这些例子，我们就能学到哪些点是占分的，哪些是不占分的，然后在考试的时候才能够回答出阅卷老师想要的答案。

BERT也是同理，虽然通过预训练，它已经有一定的能力去理解文本的含义了，但是如果我们不告诉它我们想要的是什么答案，它依然可能给出与我们想要的大相径庭的回答。这就是我们喂给BERT数据集的目的。

然后我们需要引入model和tokenizer。huggingface提供了很多种模型，每个模型都有自己的特点，我们直接调用其中的一个就可以了。

什么是tokenizer呢？具体信息大家可以去读原文章或者这篇介绍。简单来说就是BERT模型需要建立一个字典来记录可能会出现在需要理解的文本中的词汇，但是英文中存在着前缀、后缀和词根等现象，它们和一些词语的组合让需要记录的词汇数量变得非常多。比如说watch, watches和watched，他们都是一个单词watch的衍生词，但是在字典中需要占用三个词的位置来记录。所以BERT就直接把它拆成了”watch”, “##ed”和”##es”。这些元素都被成之为token，字典里只会记录token这样一来字典里需要记录的词就大幅度减少了。tokenizer就是将文本转化为token的转化器。

然后BERT中还存在有两个特殊的token：[CLS], [SEP]。这两个token是用来记录句子的起始和结束的。每个句子都会以[CLS]作为起始，以[SEP]结尾。

def question_answer(question, text):
    # tokenize question and text as a pair
    input_ids = tokenizer.encode(question, text)
    # string version of tokenized ids
    tokens = tokenizer.convert_ids_to_tokens(input_ids)
    # segment IDs
    # first occurence of [SEP] token
    sep_idx = input_ids.index(tokenizer.sep_token_id)

    # number of tokens in segment A (question)
    num_seg_a = sep_idx + 1

    # number of tokens in segment B (text)
    num_seg_b = len(input_ids) - num_seg_a

    # list of 0s and 1s for segment embeddings
    segment_ids = [0] * num_seg_a + [1] * num_seg_b

    assert len(segment_ids) == len(input_ids)

    # model output using input_ids and segment_ids
    output = model(torch.tensor([input_ids]),
            token_type_ids=torch.tensor([segment_ids]))

    # reconstructing the answer
    answer_start = torch.argmax(output.start_logits)
    answer_end = torch.argmax(output.end_logits)

    if answer_end >= answer_start:
        answer = tokens[answer_start]
        for i in range(answer_start+1, answer_end+1):
            if tokens[i][0:2] == "##":
                answer += tokens[i][2:]
            else:
                answer += " " + tokens[i]

    if answer.startswith("[CLS]"):
        answer = 'Unable to find the answer to your question.'

    print("\nPredicted answer:\n{}".format(answer.capitalize()))

这段代码的主要逻辑就是首先将我们的问题和文本转化成一个个token，然后把这些token传给语言模型进行解析，最后输出模型给出的各个答案中选取得分最高的那一个。

text = input("Please enter your text: \n")
question = input("\nPlease enter your question: \n")

while True:
    question_answer(question, text)

    flag = True
    flag_N = False

    while flag:
        response = input("\nDo you want to ask another question based on this text (Y/N)?")
        if response[0] == "Y":
            question = input("\nPlease enter your question: \n")
            flag = False
        elif response[0] == "N":
            print("\nBye!")
            flag = False
            flag_N = True

    if flag_N == True:
        break
