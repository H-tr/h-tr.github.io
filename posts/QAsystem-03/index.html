使用GPT3做简单应用并实现简易问答系统

为了简单方便，我们尝试GPT3的时候可以直接去找别人搭好的playground。将gpt3-sandbox下载下来之后我们选择其中的gpt.py来辅助我们的开发。我们假设你现在在linux环境下开发（使用Windows的同学可以尝试WSL, Windows Subsystem Linux），在terminal中输入：

1

mkdir GPTtry

建立一个文件夹作为试验田，体验GPT3的使用。在文件夹里，将gpt3-sandbox拷贝下来：

1

git clone https://github.com/shreyashankar/gpt3-sandbox.git

把其中的gpt.py留下，其余全都删除。

使用GPT的时候需要OpenAI给的secret key，这个上OpenAI的官网注册账号，然后就能拿到。建立一个GPT_SECRET_KEY.json文件，拿到secret key以后将其储存在json文件中方便我们调用：

现在我们熟悉了GPT3的调用，接下来试着完成一个简易的问答系统吧！

在OpenAI的网站上，有一个GPT3的playground。在这里我们可以调整参数，从而对与训练语言模型进行微调。微调之后可以直接导出一段代码来设定python程序中语言模型的超参数。我在其中已经尝试了一些调参，得到了一个我个人比较喜欢的结果，也希望大家能够自己玩一玩，得到自己想要的模型。现在我们先建立一个文件

import json
import openai

with open("GPT_SECRET_KEY.json") as f:
    data = json.load(f)

openai.api_key = data["API_KEY"]

def gpt3(prompt, engine='text-davinci-002', response_length=64,
         temperature=0.7, top_p=1, frequency_penalty=0, presence_penalty=0,
         start_text='', restart_text='', stop_seq=[]):
    response = openai.Completion.create(
        prompt=prompt + start_text,
        engine=engine,
        max_tokens=response_length,
        temperature=temperature,
        top_p=top_p,
        frequency_penalty=frequency_penalty,
        presence_penalty=presence_penalty,
        stop=stop_seq,
    )
    answer = response.choices[0]['text']
    new_prompt = prompt + start_text + answer + restart_text
    return answer, new_prompt

from QAsystem import gpt3

def chat():
    prompt = """Human: Hey, how are you doing?
AI: I'm good! What would you like to chat about?
Human:"""
    while True:
        prompt += input('You: ')
        answer, prompt = gpt3(prompt,
                              temperature=0.9,
                              frequency_penalty=1,
                              presence_penalty=1,
                              start_text='\nAI:',
                              restart_text='\nHuman: ',
                              stop_seq=['\nHuman:', '\n'])
        print('GPT-3:' + answer)


if __name__ == '__main__':
    chat()

其中prompt是传入语言模型的初始对话，它可以让语言模型知道接下来应该按照什么样的模式进行对话和问答。然后我们每次输入的话都应该不断地添加在prompt后面，这样语言模型才能够记住之前有过什么对话，从而不会出现明显的割裂和违和感。

1
